# Security Policy

## Supported Versions

Only the latest `main` branch and tagged releases are currently supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do NOT open a public issue. Instead, please report it via one of the following methods:

  - **Email**: Reach out to the maintainers (see GitHub profile).
  - **GitHub Security**: Use the "Report a vulnerability" button in the "Security" tab.

We will acknowledge your report within 48 hours and provide a timeline for a fix and public disclosure.
